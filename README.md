# 项目简介

## 核心功能
- 基于YOLOv8的电梯扶梯危险检测系统，支持行李箱、倒下、站立、弯曲四类目标识别
- 提供完整的源代码、标注数据集（2600张）、训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件及本地摄像头
- 界面功能：实时显示目标位置、目标总数、置信度等信息；支持批量检测和结果保存
- 可自定义检测参数：置信度阈值、IOU阈值，可选择显示标签和原图，支持多模型切换

## 适用场景
- 电梯扶梯安全监控与管理
- 公共场所安全风险预警
- 智能安防系统集成
- 人员行为异常检测研究

## 技术特点
- 开发环境：Pycharm+Anaconda或VSCode+Anaconda
- 技术栈：Python3.9、OpenCV-Python、PyQt5、Ultralytics、PyTorch1.9等
- 采用YOLOv8目标检测算法，具备较高的检测精度和实时性
- 提供完整的用户界面，操作简便，适合部署于不同场景
